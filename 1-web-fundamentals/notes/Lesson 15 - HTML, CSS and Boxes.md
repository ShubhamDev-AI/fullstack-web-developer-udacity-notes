# HTML, CSS and Boxes

## HTML

## The box model
- Every box contains 4 area: Content, Padding, Border, and Margin
- To fit css into many browser use this:
```css
*{
    --webkit-box-sizing: border-box;
    --moz-box-sizing: border-box;
    --ms-box-sizing: border-box;
    box-sizing: border-box;
}
```
